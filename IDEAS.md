# IDEAS: VST Monster Future & Pivots

- **AI Tagging**: Use machine learning to analyze plugin descriptions and automatically tag them with genres/moods.
- **Cloud Sync**: Sync your plugin collection across multiple machines.
- **Yabridge Integration**: Automatically set up Windows VSTs on Linux using yabridge.
- **Community Reviews**: Built-in rating and review system for plugins.
- **Direct DAW Integration**: Plugins that help browse the VST Monster catalog from within the DAW.
- **Client Caching Layer**: Add a local state cache (via Redis or internal Svelte stores) to prevent over-fetching the /plugins endpoint on each UI refresh.
